# 瑞吉外卖流程和遇到的问题

      - 登录验证模块
        - 每次重新启动都需要重新判断用户是否已登陆
      - 登录注销模块
        - 问题：用户不进行登录也可以看到管理员页面
        - 解决办法：使用过滤器进行判断
      - 分页查询
        - 使用mybatisPlus完成工作
      - 新增员工信息
        - 问题：重复新增同用户名时会报错
        - 解决：ExceptionHandler
      - 员工权限模块
        - 更改status值
        - 遇到问题：Long型进度缺失，导致传回id错误
          - js的long进度只有16位
          - 解决方法：把long转换成string  ===》拓展springmvc的消息转换器
      - 编辑员工信息
    - 菜品分类管理
      - 使用公共字段进行填充
    - 文件上传下载
      - 问题：默认保存在临时目录，请求结束后就会删除
      	- 上传的形参名字必须为file
    - 菜品分类分页展示
      - 问题：菜品类型无法显示，原有的类中没有菜品类型信息
      - 解决：创建新的类来复制原有对象的xing'x
